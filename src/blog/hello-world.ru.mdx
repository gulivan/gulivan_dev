## Блоки кода

И конечно, подсветка синтаксиса:

```typescript
function greet(name: string): string {
  return `Привет, ${name}! Добро пожаловать в блог.`;
}

console.log(greet('Мир'));
```

На этом демо завершено. Следите за новыми постами!
